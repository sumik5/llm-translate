# Changelog

このプロジェクトの主要な変更履歴です。

## [v1.0.1] - 2025-08-26

### 変更
- JavaScriptをmin化
- ヘッダー構造を新しいレイアウトに更新（翻訳ボタンとHTMLを保存ボタンをヘッダーに配置）

### 追加
- LLM翻訳ツールの初回リリース
- Markdown/EPUB/PDF対応
- 多言語翻訳機能（日本語、英語、中国語、韓国語、スペイン語、フランス語、ドイツ語、ロシア語、ポルトガル語、イタリア語）
- LM Studioのローカルモデルとの連携
- チャンク分割処理による大規模ドキュメント対応
- 翻訳中の中止機能
- Markdownプレビュー機能
- コードブロックの構文ハイライト表示
